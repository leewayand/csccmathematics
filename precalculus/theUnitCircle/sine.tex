If we watch how the $y$-coordinates change as the point travels counterclockwise around the unit circle, then we will see exactly how the sine function behaves.






\subsection*{Core Sine Function}



The core sine function is $\sin(\theta)$.

Its characteristics exactly follow the characteristics of the $y$-coordinates of points on the unit circle.






\textbf{\textcolor{blue!55!black}{Domain}}

The domain of $\sin(\theta)$ is $(-\infty, \infty)$.

A point can be rotated counterclockwise (positive angles) or clockwise (negative angles) any number of rotations.



\textbf{\textcolor{blue!55!black}{Zeros}}


The zeros of $\sin(\theta)$ are the angles where the $y$-coordinate equals $0$.  These are the angles which point in the direction of the points $(1,0)$ and $(-1,0)$.

That would be an infinite collection of angles.

\[
\{ \, k \, \pi \, \text{ | } \,  k \in \mathbb{Z} \,  \}
\]



\textbf{\textcolor{blue!55!black}{Continuity}}


$\sin(\theta)$ is a continuous function with no singularities.




\textbf{\textcolor{blue!55!black}{End-Behavior}}



Since sine is the $y$-coordinate of points on the unit circle, it just keeps oscillating between $-1$ and $1$ as the angle rotates.

$\sin(\theta)$ has no end-behavior other than oscillating.




\textbf{\textcolor{blue!55!black}{Behavior}}


Following the $y$-coordinates for one cycle around the unit circle, we can see the full behavior of $\sin(\theta)$.



\begin{itemize}
\item On $\left( 0, \frac{\pi}{2} \right)$, $\sin(\theta)$ increases from $0$ to $1$.
\item On $\left( \frac{\pi}{2}, \pi \right)$, $\sin(\theta)$ decreases from $1$ to $0$.
\item On $\left( \pi, \frac{3\pi}{2} \right)$, $\sin(\theta)$ decreases from $0$ to $-1$.
\item On $\left( \frac{3\pi}{2}, 2\pi \right)$, $\sin(\theta)$ increases from $-1$ to $0$.
\end{itemize}


Since $\sin(\theta)$ is periodic with period $2\pi$, this behavior keeps repeating.





\textbf{\textcolor{blue!55!black}{Global Maximum and Minimum}}



$\sin(\theta)$ is a continuous function and switches from increasing to decreasing at $\frac{\pi}{2}$.  


That makes $\sin\left( \frac{\pi}{2} \right) = 1$ the global maximum, which occurs at $\frac{\pi}{2}$.


Since $\sin(\theta)$ is periodic with period $2\pi$, this maximum keeps occurring.


The maximum value of $1$ occurs on the set 

\[
\left\{ \, \frac{\pi}{2} + 2k \pi \, \text{ | } \,  k \in \mathbb{Z} \,  \right\}
\]







$\sin(\theta)$ is a continuous function and switches from decreasing to increasing at $\frac{3\pi}{2}$.  


That makes $\sin\left( \frac{3\pi}{2} \right) = 1$ the global minimum, which occurs at $\frac{3\pi}{2}$.




Since $\sin(\theta)$ is periodic with period $2\pi$, this minimum keeps occurring.


The maximum value of $-1$ occurs on the set 

\[
\left\{ \, \frac{3\pi}{2} + 2k \pi \, \text{ | } \,  k \in \mathbb{Z} \,  \right\}
\]








\textbf{\textcolor{blue!55!black}{Local Maximum and Minimum}}



The global extrema are automatically local extrema and they are the only ones, since there are no other critical numbers (switching of behavior).







\textbf{\textcolor{blue!55!black}{Range}}


$\sin(\theta)$ is a continuous function with a maximum of $1$ and a minimum of $-1$.

Therefore, the range is $[-1,1]$.






\begin{observation} \textbf{\textcolor{-red!75}{Sine is the $y$-coordinate}}

The idea is that the core sine function describes the $y$-coordinate as you travel counterclockwise around the unit circle.

The characteristics of the core sine function hold steady during each quadrant, changing at the angles $0$, $\frac{\pi}{2}$, $\pi$, $\frac{3\pi}{2}$, and $2\pi$. From there, they keep repeating.

This angle is described by the expression inside the parentheses in $\sin(angle)$.


\end{observation}





We can build more general sine functions as we have been doing, through compositions.



\begin{idea} \textbf{\textcolor{blue!55!black}{General Sine Functions as Compositions}} 


\begin{itemize}
\item Let $L_{out}(u) = A \, u + D$, a linear function
\item Let $S(\theta) = \sin(\theta)$, the core sine function
\item Let $L_{in}(v) = B \, v + C$, a linear function
\end{itemize}


\[
A \, \sin(B \, t + C) + D = (L_{out} \circ S \circ L_{in})(t)
\]



\end{idea}







\begin{example} Behavior Sine

Describe the behavior of   $f(x) = \sin\left( 2 x + \frac{\pi}{2} \right)$



\textbf{\textcolor{red!75!green}{explanation}} 






\begin{itemize}
\item Let $L_{out}(u) = u$, an increasing linear function
\item Let $S(\theta) = \sin(\theta)$, the core sine function
\item Let $L_{in}(v) = 2 \, v + \frac{\pi}{2}$, an increasing linear function
\end{itemize}


\[
f(x) = \sin\left( 2 x + \frac{\pi}{2} \right) = (L_{out} \circ S \circ L_{in})(x)
\]


The core sine funtion increases and decreases as the angle passes through the four quadrants.


The ``angle'' is $2 x + \frac{\pi}{2}$.

We want to know when $2 x + \frac{\pi}{2}$ has the values $0$, $\frac{\pi}{2}$, $\pi$, $\frac{3\pi}{2}$, and $2\pi$.






\[
\begin{array}{l|l}
\text{angle}                  & x  \\ \hline
2 x + \frac{\pi}{2} = 0        & x = -\frac{\pi}{4}  \\
2 x + \frac{\pi}{2} = \frac{\pi}{2}      &  x = 0 \\
2 x + \frac{\pi}{2} = \pi      & x = \frac{\pi}{4}  \\
2 x + \frac{\pi}{2} = \frac{3\pi}{2}     & x = \frac{\pi}{2}   \\
2 x + \frac{\pi}{2} = 2\pi     & x = \frac{3\pi}{4}
\end{array}
\]


We now know which values of $x$ define our quadrants.





\[
\begin{array}{l|l}
\text{Quadrant}                  & x\text{-Interval}  \\  \hline
I        & x \in  \left( -\frac{\pi}{4}, 0 \right)  \\
II       & x \in  \left( 0, \frac{\pi}{4} \right) \\
III      & x \in  \left( \frac{\pi}{4}, \frac{\pi}{2} \right)  \\
IV       & x \in  \left( \frac{\pi}{2}, \frac{3\pi}{4} \right)   \\
\end{array}
\]



Our principal interval is $\left( -\frac{\pi}{4}, \frac{3\pi}{4} \right)$. These are the values of $x$ that make a full circle of angles.

That makes the period $\frac{3\pi}{4} - \left( -\frac{\pi}{4} \right) = \pi$.



Since both linear functions in our composition are increasing linear functions, the behavior of the whole composition will be the same as the behavior of sine.

Now we just remember what the core sine function does in each quadrant. 



\textbf{\textcolor{blue!55!black}{Behavior}} 



\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (I):}} On $\left( -\frac{\pi}{4}, 0 \right)$, $f(x) = inc \circ inc \circ inc$.  $f$ increases from $0$ to $1$.



\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (II):}} On $\left( 0, \frac{\pi}{4} \right)$, $f(x) = inc \circ dec \circ inc$. $f$ decreases from $1$ to $0$.



\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (III):}} On $\left( \frac{\pi}{4}, \frac{\pi}{2} \right)$, $f(x) = inc \circ dec \circ inc$ . $f$ decreases from $0$ to $-1$.


\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (IV):}} On $\left( \frac{\pi}{2}, \frac{3\pi}{4} \right)$, $f(x) = inc \circ inc \circ inc$. $f$ increases from $-1$ to $0$.



All of this repeats with a period of $\pi$.

\textbf{\textcolor{blue!55!black}{Extrema}} 


On our principal interval, $f(x)$ has a global and local maximum of $1$ at $0$.


On our principal interval, $f(x)$ has a global and local minimum of $-1$ at $\frac{\pi}{2}$.




\textbf{\textcolor{blue!55!black}{Range}} 

Since sine is continuous, the extreme values give us a range of $[-1,1]$.



\end{example}




\begin{warning} \textbf{\textcolor{red!70!black}{Variable vs Angle}}


We must distinguish between the variable of the function and the angle of sine.

The variable in $f(x) = \sin\left( 2 x + \frac{\pi}{2} \right)$ is $x$.

That is not the angle.

The angle is $2 x + \frac{\pi}{2}$.

Our analysis follows the angle around the unit circle through the four quadrants.

\end{warning}

























\begin{example} Negative Leading Coefficient Inside

Describe the behavior of   $g(t) = \sin\left( \frac{\pi}{4} - 3 t \right)$



\textbf{\textcolor{red!75!green}{explanation}} 







\begin{itemize}
\item Let $L_{out}(u) = u$, an increasing linear function
\item Let $S(\theta) = \sin(\theta)$, the core sine function
\item Let $L_{in}(v) = \frac{\pi}{4} - 3 t$, a decreasing linear function
\end{itemize}


\[
g(t) = \sin\left( \frac{\pi}{4} - 3 t \right) = (L_{out} \circ S \circ L_{in})(t)
\]





The ``angle'' is $\frac{\pi}{4} - 3 t$.

We want to know when $\frac{\pi}{4} - 3 t$ has the values $0$, $\frac{\pi}{2}$, $\pi$, $\frac{3\pi}{2}$, and $2\pi$.






\[
\begin{array}{l|l}
\text{angle}                  & t  \\ \hline
\frac{\pi}{4} - 3 t = 0        & t = \frac{\pi}{12}  \\
\frac{\pi}{4} - 3 t = \frac{\pi}{2}      &  t = -\frac{\pi}{12} \\
\frac{\pi}{4} - 3 t = \pi      & t = -\frac{\pi}{4}  \\
\frac{\pi}{4} - 3 t = \frac{3\pi}{2}     & t = -\frac{5\pi}{12}   \\
\frac{\pi}{4} - 3 t = 2\pi     & t = -\frac{7\pi}{12}
\end{array}
\]


We now know which values of $t$ define our quadrants. 




\[
\begin{array}{l|l}
\text{Quadrant}                  & t\text{-Interval}  \\ \hline
I        & t \in  \left( -\frac{\pi}{12}, \frac{\pi}{12} \right)  \\
II       & t \in  \left( -\frac{\pi}{4}, -\frac{\pi}{12} \right) \\
III      & t \in  \left( -\frac{5\pi}{12}, -\frac{\pi}{4} \right)  \\
IV       & t \in  \left( -\frac{7\pi}{12}, -\frac{5\pi}{12} \right)   \\
\end{array}
\]

They are running backwards.






Our principal interval is $\left( -\frac{7\pi}{12}, \frac{\pi}{12} \right)$.

That makes the period $\frac{\pi}{12} - \left( -\frac{7\pi}{12} \right) = \frac{\pi}{12} -\frac{7\pi}{12} = \frac{8\pi}{12} = \frac{2\pi}{3}$.


Now we just remember what the basic sine function does in each quadrant. 







\textbf{\textcolor{blue!55!black}{Behavior}} 



\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (I):}} On $\left( -\frac{\pi}{12}, \frac{\pi}{12} \right)$, $inc \circ inc \circ dec = dec$ : $g(t)$ decreases from $0$ to $-1$.



\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (II):}} On $\left( -\frac{\pi}{4}, \frac{\pi}{4} \right)$, $inc \circ dec \circ dec = inc$ : $g(t)$ increases from $-1$ to $0$.



\textbf{\textcolor{blue!55!black}{$\blacktriangleright$(III):}} On $\left( -\frac{5\pi}{12}, -\frac{\pi}{4} \right)$, $inc \circ dec \circ dec = inc$ : $g(t)$ increases from $0$ to $1$.


\textbf{\textcolor{blue!55!black}{$\blacktriangleright$(IV):}} On $\left( -\frac{7\pi}{12}, -\frac{5\pi}{12} \right)$, $inc \circ inc \circ dec = dec$ : $g(t)$ decreases from $1$ to $0$.



All of this repeats with a period of $\frac{2\pi}{3}$.

\textbf{Note:}  In the example above, the inside of the sine function is a linear function with a negative leading coefficient.  That has reversed the behavior from the core sine function.




\textbf{\textcolor{blue!55!black}{$\blacktriangleright$}} In addition to the leading coefficient of the inner linear function, the sine function itself has a leading coefficient.  When it is negative, the behavior reverses again.




The example above illustrates a fact about function behavior.



\subsection*{Composition and Behavior}

Composition with decreasing functions reverses behavior.

\begin{fact}

If $f(x)$ is increasing on $(a, b)$, then $f(-x)$ is decreasing on $(a, b)$.

\textbf{\textcolor{red!75!green}{explanation}} 

$f(x)$ is increasing on $(a, b)$ means that for any $x$ and $y$ in $(a, b)$ with $x < y$, it must be true that $f(x) < f(y)$.


We want to show that $f(-x)$ is decreasing on $(a, b)$.  Let's call $f(-x)$ by a new name.  $g(x) = f(-x)$

We want to show that $g(x)$ is decreasing.

We want to show that that for any $x$ and $y$ in $(a, b)$ with $x < y$, it must be true that $g(x) > g(y)$.



So, let's pretend we have $x$ and $y$ in $(a, b)$ with $x < y$.

Then $-x > -y$.

Then $f(-x) > f(-y)$, because $f$ is increasing.

But that is the same as $g(x) > g(y)$.

That means that $g$ is decreasing.


\[ g(x) = (f \circ -x) \]

\begin{center}
$g$ has the opposite behavior as $f$
\end{center}

\end{fact}















This fact is connected to another fact about function behavior.




\begin{fact}

If $f(x)$ is increasing on $(a, b)$, then $-f(x)$ is decreasing on $(a, b)$.

\textbf{\textcolor{red!75!green}{explanation}} 

$f(x)$ is increasing on $(a, b)$ means that for any $x$ and $y$ in $(a, b)$ with $x < y$, it must be true that $f(x) < f(y)$.


We want to show that $-f(x)$ is decreasing on $(a, b)$.  Let's call $-f(x)$ by a new name.  $g(x) = -f(x)$

We want to show that $g(x)$ is decreasing.

We want to show that that for any $x$ and $y$ in $(a, b)$ with $x < y$, it must be true that $g(x) > g(y)$.



So, let's pretend we have $x$ and $y$ in $(a, b)$ with $x < y$.

Then $f(x) < f(y)$, because $f$ is increasing.

Then $-f(x) > -f(y)$.

But that is the same as $g(x) > g(y)$.

That means that $g$ is decreasing.


\[ g(x) = (-x \circ f) \]

\begin{center}
$g$ has the opposite behavior as $f$
\end{center}


\end{fact}




\textbf{Idea:} Compositing with linear functions that have negative leading coefficients reverse behavior.








\begin{example} Negative Leading Coefficient

Describe the behavior of   $H(k) = -2 \sin\left( \frac{k}{3} - \pi \right)$



\textbf{\textcolor{red!75!green}{explanation}} 



\begin{itemize}
\item Let $L_{out}(u) = -2 \, u$, a decreasing linear function.
\item Let $S(\theta) = \sin(\theta)  \, u$, the core sine function.
\item Let $L_{in}(v) = \frac{u}{3} - \pi$, an increasing linear function.
\end{itemize}



\[
H(k) = -2 \sin\left( \frac{k}{3} - \pi \right) = (L_{out} \circ S \circ L_{in})(k)
\]



The ``angle'' is $\frac{k}{3} - \pi$.

We want to know when $\frac{k}{3} - \pi$ has the values $0$, $\frac{\pi}{2}$, $\pi$, $\frac{3\pi}{2}$, and $2\pi$.






\[
\begin{array}{l|l}
\text{angle}                  & k  \\ \hline
\frac{k}{3} - \pi = 0        & k = 3 \pi  \\
\frac{k}{3} - \pi = \frac{\pi}{2}      &  k = \frac{9\pi}{2} \\
\frac{k}{3} - \pi = \pi      & k = 6 \pi  \\
\frac{k}{3} - \pi = \frac{3\pi}{2}     & k = \frac{15\pi}{2}   \\
\frac{k}{3} - \pi = 2\pi     & k = 9 \pi
\end{array}
\]


We now know which values of $k$ define our quadrants.





\[
\begin{array}{l|l}
\text{Quadrant}                  & k\text{-Interval}  \\  \hline
I        & k \in  \left( 3\pi, \frac{9\pi}{2} \right)  \\
II       & k \in  \left( \frac{9\pi}{2}, 6 \pi \right) \\
III      & k \in  \left( 6 \pi, \frac{15\pi}{2} \right)  \\
IV       & k \in  \left( \frac{15\pi}{2}, 9 \pi \right)   \\
\end{array}
\]



Our principal interval is $\left( 3 \pi, 9 \pi \right)$.

That makes the period $9 \pi - 3 \pi = 6 \pi$.



The global maximum and minimum values of the core sine function are $-1$ and $1$. When the compositoin passes these into $L_{out}(u) = -2 \, u$, we will get the values $-2$ and $2$.




Now we just remember what the basic sine function does in each quadrant. 



\textbf{\textcolor{blue!55!black}{Behavior}} 



\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (I):}} On $\left( 3 \pi, \frac{9\pi}{2} \right)$, $dec \circ inc \circ inc = dec$ : $f(k)$ decreases from $0$ to $-2$.



\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (II):}} On $\left( \frac{9\pi}{2}, 6 \pi \right)$, $dec \circ dec \circ inc = inc$ : $f(k)$ increases from $-2$ to $0$.



\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (III):}} On $\left( 6 \pi, \frac{15\pi}{2} \right)$, $dec \circ dec \circ inc = inc$ : $f(k)$ increases from $0$ to $2$.


\textbf{\textcolor{blue!55!black}{$\blacktriangleright$ (IV):}} On $\left( \frac{15\pi}{2}, 9 \pi \right)$, $dec \circ inc \circ inc = dec$ : $f(k)$ decreases from $2$ to $0$.



All this repeats with a period of $6 \pi$.

\begin{formula} \textbf{\textcolor{blue!55!black}{Core Sine Function}}

The \textbf{Core Sine Function} is any function that \textbf{\textcolor{purple!85!blue}{CAN}} be represented with the formula 

\[     S(x) =    \sin(x)           \]




\end{formula}











\begin{formula} \textbf{\textcolor{blue!55!black}{General Sine Functions}}

A \textbf{General Sine Function} is any function that \textbf{\textcolor{purple!85!blue}{CAN}} be represented with a formula of the form

\[     S(x) =    A \sin(B \, x + C) + D           \]

where $A$, $B$, $C$, and $D$ are real numbers, and $A, B \ne 0$.


This can be viewed as a composition.




\begin{itemize}
\item Let $L_{out}(u) = A \, u + D$, a linear function.
\item Let $S(\theta) = \sin(\theta)$, the core sine function.
\item Let $L_{in}(v) = B \, v + C$, a linear function.
\end{itemize}

\[     S(x) = A \sin(B \, x + C) + D = (L_{out}(u) \circ S \circ L_{in})(x)          \]


The Chain Rule will now give us the bhavior.


\end{formula}
