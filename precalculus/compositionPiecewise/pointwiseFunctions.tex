Evaluating compositions of piecewise functions is a four step process.

To evaluate $F(G(a))$,


\textbf{\textcolor{red!75!green}{First,}} locate which interval of the domain of $G$ includes $a$. There is only one such interval. 


\textbf{\textcolor{red!75!green}{Second,}} the identified interval has exactly one formula associated with it. Use that formula to evaluate $G(a)$.


\textbf{\textcolor{red!75!green}{Third,}} locate which interval of the domain of $F$ includes $G(a)$. There is only one such interval. 

\textbf{\textcolor{red!75!green}{Fourth,}} the identified interval has exactly one formula associated with it. Use that formula to evaluate $F(G(a))$.
